# 7. Handlers

## Resposta

```yaml
- hosts: webservers
  tasks:
    - name: Atualizar arquivo de configuração
      copy:
        src: apache.conf
        dest: /etc/apache2/apache2.conf
      notify: Reiniciar Apache

  handlers:
    - name: Reiniciar Apache
      service:
        name: apache2
        state: restarted
```

## Explicação
Handlers são executados somente quando notificados, otimizando execuções.
